PROJECT DESCRIPTION
Stock Price Predictor, the  webScraper scrapes the (user determined) yahoo finance page for the current stock price. The program executes this action 10 times over a (user determined) amount of time. The dataProcessor class will use this data to run a linear regression to predict the next determined stock price.

BeautifulSoup
Pandas
Scikit-learn
Time


HOW TO RUN IT
As long as you have access to all of the classes, you should be able to run the project. run Main().py, input desired time interval between data collection points, and the ticker symbol of the desired company.
